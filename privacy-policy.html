1. Pendahuluan

Selamat datang di Kamus Korea. Kami berkomitmen untuk melindungi privasi Anda. Kebijakan Privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi informasi pribadi Anda saat menggunakan aplikasi mobile Kamus Korea (selanjutnya disebut "Aplikasi").

Dengan menggunakan Aplikasi ini, Anda menyetujui pengumpulan dan penggunaan informasi sesuai dengan kebijakan ini.

2. Informasi yang Kami Kumpulkan

2.1. Informasi Akun

Saat Anda mendaftar atau masuk menggunakan Google Sign-In, kami mengumpulkan:

  • Nama: Nama lengkap dari akun Google Anda
  • Email: Alamat email untuk identifikasi akun
  • Foto Profil: Foto profil dari akun Google Anda (opsional)
  • User ID: ID unik yang dihasilkan oleh Firebase Authentication

2.2. Data Pembelajaran

Kami mengumpulkan data aktivitas pembelajaran Anda untuk meningkatkan pengalaman belajar:

  • Kata-kata favorit yang Anda simpan
  • Riwayat pencarian kata di kamus
  • Hasil quiz dan penilaian
  • Progress pembelajaran (XP, level, achievement)
  • Streak harian (hari berturut-turut belajar)
  • Statistik penggunaan aplikasi

2.3. Data Teknis

Kami mengumpulkan informasi teknis untuk analisis dan perbaikan aplikasi:

  • Model perangkat dan versi Android
  • Alamat IP dan informasi jaringan
  • Log aktivitas aplikasi
  • Data crash dan error reports (via Firebase Crashlytics)
  • Waktu penggunaan aplikasi

2.4. Data Iklan

Aplikasi menggunakan Google AdMob untuk menampilkan iklan. Data yang dikumpulkan meliputi:

  • Advertising ID (untuk personalisasi iklan)
  • Interaksi dengan iklan (klik, impressions)
  • Data perangkat untuk penargetan iklan

3. Bagaimana Kami Menggunakan Informasi Anda

Informasi yang kami kumpulkan digunakan untuk:

3.1. Penyediaan Layanan

  • Membuat dan mengelola akun pengguna
  • Menyinkronkan data pembelajaran antar perangkat
  • Menyimpan progress dan preferensi Anda
  • Menampilkan konten yang dipersonalisasi

3.2. Peningkatan Pengalaman

  • Menganalisis pola penggunaan untuk meningkatkan fitur
  • Memperbaiki bug dan masalah teknis
  • Mengoptimalkan performa aplikasi
  • Mengembangkan fitur baru berdasarkan kebutuhan pengguna

3.3. Gamifikasi

  • Menghitung XP, level, dan achievement
  • Menampilkan leaderboard dan ranking
  • Memberikan reward dan notifikasi pencapaian
  • Tracking quest harian dan streak

3.4. Komunikasi

  • Mengirim notifikasi push (reminder belajar, achievement unlock)
  • Mengirim update penting terkait aplikasi
  • Merespon pertanyaan dan dukungan pelanggan

4. Berbagi Informasi dengan Pihak Ketiga

Kami TIDAK menjual informasi pribadi Anda kepada pihak ketiga. Namun, kami berbagi data dengan layanan berikut:

4.1. Firebase (Google)

  • Firebase Authentication: Untuk login dan autentikasi
  • Firebase Analytics: Untuk analisis penggunaan aplikasi
  • Firebase Cloud Messaging: Untuk push notifications
  • Firebase Crashlytics: Untuk monitoring crash (jika diaktifkan)

Informasi lebih lanjut: Firebase Privacy Policy

4.2. Google AdMob

  • Menampilkan iklan yang relevan
  • Mengumpulkan data untuk personalisasi iklan

Informasi lebih lanjut: Google Privacy Policy

4.3. Server Backend Kami

  • Menyimpan data pembelajaran di server (webtechsolution.my.id)
  • Sinkronisasi XP, level, achievement, dan leaderboard
  • Data dienkripsi menggunakan HTTPS

5. Keamanan Data

Kami menerapkan langkah-langkah keamanan untuk melindungi data Anda:

  • Enkripsi HTTPS: Semua komunikasi dengan server dienkripsi
  • Firebase Authentication: Autentikasi yang aman dengan token
  • Penyimpanan Lokal Terenkripsi: Data sensitif disimpan dengan DataStore
  • Akses Terbatas: Hanya developer yang berwenang yang dapat mengakses database
  • Regular Security Updates: Aplikasi diperbarui secara berkala untuk keamanan

6. Hak Pengguna

Anda memiliki hak berikut terkait data pribadi Anda:

6.1. Akses Data

Anda dapat melihat data pribadi Anda di dalam aplikasi (Profile ‚Üí Pengaturan).

6.2. Koreksi Data

Anda dapat mengubah nama dan foto profil melalui menu Profile di aplikasi.

6.3. Penghapusan Data

Anda dapat menghapus akun dan semua data terkait dengan cara:

  • Buka aplikasi ‚Üí Profile ‚Üí Pengaturan
  • Pilih "Hapus Akun"
  • Konfirmasi penghapusan

Atau hubungi kami di email support untuk permintaan penghapusan manual.

6.4. Opt-Out dari Iklan Personalisasi

Anda dapat menonaktifkan iklan personalisasi melalui:

  • Android: Settings ‚Üí Google ‚Üí Ads ‚Üí Opt out of Ads Personalization
  • Atau kunjungi: Google Ads Settings

6.5. Opt-Out dari Notifikasi

Anda dapat menonaktifkan push notifications melalui:

  • Aplikasi ‚Üí Pengaturan ‚Üí Notifikasi ‚Üí OFF
  • Atau melalui Android Settings ‚Üí Apps ‚Üí Kamus Korea ‚Üí Notifications ‚Üí OFF

7. Data Anak-Anak

Aplikasi ini tidak secara khusus ditujukan untuk anak-anak di bawah usia 13 tahun. Kami tidak secara sengaja mengumpulkan informasi pribadi dari anak-anak di bawah 13 tahun. Jika Anda adalah orang tua dan mengetahui bahwa anak Anda memberikan informasi pribadi kepada kami, silakan hubungi kami agar kami dapat menghapus data tersebut.

8. Cookies dan Teknologi Tracking

Aplikasi menggunakan teknologi tracking untuk meningkatkan pengalaman pengguna:

  • Firebase Analytics: Untuk menganalisis penggunaan aplikasi
  • AdMob Cookies: Untuk menampilkan iklan yang relevan
  • Local Storage: Untuk menyimpan preferensi dan data sesi

Anda dapat menonaktifkan tracking melalui pengaturan perangkat Anda.

Data Anda mungkin ditransfer dan disimpan di server yang berlokasi di Indonesia dan/atau negara lain tempat Firebase/Google beroperasi. Dengan menggunakan aplikasi ini, Anda menyetujui transfer data tersebut.

Kami menyimpan data Anda selama:

  • Data Akun: Selama akun Anda aktif
  • Data Pembelajaran: Selama akun Anda aktif

Setelah akun dihapus, data Anda akan dihapus secara permanen dalam 30 hari.
